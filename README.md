# StudentManager

**StudentManager** — bu C++ tilida yozilgan mini loyiha bo‘lib, talaba ma’lumotlarini boshqarish uchun mo‘ljallangan. Loyiha **OOP**, **vector**, **pointer/reference**, va **constructor/initializer list** kabi asosiy C++ tushunchalarini amalda qo‘llashga xizmat qiladi.

## Asosiy funksiyalar

- Student struct (OOP)  
- Constructor va initializer list bilan obyekt yaratish  
- `tanishtir()`, `yosh()`, `bosqich()`, `katta()` kabi methodlar  
- Vector ichida Student obyektlarini saqlash va boshqarish  
- Reference va pointer yordamida vector elementlarini tezkor va xavfsiz ishlatish

--------------
Windows
```
g++ -std=c++20 main.cpp -o StudentManager

clang++ -std=c++20 main.cpp -o StudentManager
```
3. Dastur ishga tushuring:
```
./StudentManager
```
